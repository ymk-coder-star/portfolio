# Portfolio

Personal portfolio site for **Yisroel Krausz** — junior frontend developer. Showcases featured projects, case studies, tech stack, and contact links.

**Live site:** [codecanvas.uk](https://codecanvas.uk)

## Features

- Project gallery with featured and additional work
- Case study pages for selected builds
- About page with bio, tech stack, and contact links
- Responsive layout with subtle motion (respects `prefers-reduced-motion`)
- Optimized WebP assets, custom favicon, and Open Graph meta tags

## Tech stack

- React 19
- React Router
- Tailwind CSS
- Create React App
- Sharp (image / favicon generation)
- Vercel

## Getting started

Clone the repo, install dependencies, and start the dev server:

```bash
git clone https://github.com/ymk-coder-star/portfolio.git
cd portfolio
npm install
npm start
```

Runs the app at [http://localhost:3000](http://localhost:3000).

## Project structure

```
src/
  components/   # Layout and reusable UI
  data/         # Projects, case studies, tech stack
  hooks/        # Custom hooks (e.g. hero parallax)
  pages/        # Home, About, Case Study
  assets/       # Images and monogram mark
  constants/    # Site metadata (URL, titles, resume)
```
